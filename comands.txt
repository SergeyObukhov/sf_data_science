Установить NumPy
- выбираем среду разработки, куда будем устанавливать;
    pip install numpy

Зафиксировать версии библиотеки, которые использовались в решении (проекте).

Несколько подходов.

С помощью PIP
1) зафиксировать библиотеки pip:
    - pip freeze > requirements.txt
чтобы потом установить библиотеки из файла requirements.txt:
    - install -r requirements.txt
2) сохранить версию питона и библиотеки:
    - conda env export > environment.yml
чтобы потом установить (полностью создается окружение - копия среды):
    - conda env create -f environment.yml

Более продвинутый способ управления пакетами:
    - Poetry
    - Docker (для изоляции окружения)
